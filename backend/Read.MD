# Technavyug Backend

A secure Node.js + Express backend for the Technavyug platform featuring:

* JWT Authentication
* MongoDB Integration
* Admin Dashboard APIs
* Projects Management
* Certificates Management
* Achievements Management
* RESTful APIs

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcrypt.js

## Features

* Admin Authentication
* CRUD APIs
* Secure Protected Routes
* MongoDB Atlas Integration
* Certificate Verification APIs
* Achievement APIs
* Projects APIs

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```env
PORT=5001
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
